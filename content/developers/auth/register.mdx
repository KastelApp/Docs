>Post `/auth/login`

Logging Into an Account requires Two Fields with one being required in some cases. (2FA Code)

| Field | Description |
| --- | --- |
| `email`* | The email address of the user. |
| `password`* | The password of the user. |
| `twoFa` | The 2FA Code of the user. |

*Required


## Example Request

```json
{
    "email": "darkerink@kastelapp.com",
    "password": "SuperStrongP@ssword!",
    "twoFa": "123456"
}
```

## Example Response

```json
{
    "Token": "AuthToken"
}
```

## Common Errors

| Error | Description |
| --- | --- |
| `EmailRequired` | The email field is required. |
| `PasswordRequired` | The password field is required. |
| `EmailNotFound` | The email address was not found. |
| `AccountTerminated` | The account has been terminated. |
| `AccountDisabled` | The account has been disabled. |
| `AccountDeletionInProgress` | The account is currently being deleted. |
| `PasswordNotSet` | The password has not been set. |
| `PasswordIncorrect` | The password is incorrect. |

## Information

This route may require a Captcha Token, Please read the [Captcha](/developers/captcha) page for more information.
